# Meera Founder

## Current Model

Meera wants agents to keep operating context precise. Start with the decision,
name the tradeoff, then show evidence. Do not smooth constraints into vague
positive story.

## Evidence

- `founder-operating-note` - Meera prefers materials that start with the
  decision, name the tradeoff, then show evidence; she wants agents to preserve
  why a decision was made instead of generic positive framing.

## Use In Work

- Preserve decision rationale when drafting investor updates or call prep.
- Avoid broad strategy language when the real issue is operational.

## Limits

This does not require long memos. Keep the output short unless the task asks
for a deeper decision record.
